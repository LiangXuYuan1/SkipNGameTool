# SkipNGameTool 逢N过游戏工具

大家聚会的时候可能会玩过一个游戏：逢6过
游戏规则：从1开始轮流报数，当要报的数字包含6或者是6的倍数的时候要说“过”。喊错的人就输了，输了就要接受惩罚。
这个工具可以把要喊的数或过列出来，这样就不怕啦。
